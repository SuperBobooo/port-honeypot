# 实训报告模板

## 1. 项目名称

轻量跨平台端口蜜罐开发

## 2. 项目背景

说明端口探测、内网横向移动、低交互蜜罐的安全价值，以及本项目在企业内网安全监测中的作用。

## 3. 需求分析

- 轻量化客户端。
- 跨平台部署。
- 多端口诱捕。
- 加密日志上传。
- 服务端统一管理。
- SQLite 存储与日志检索。
- 多通道告警。

## 4. 总体设计

描述 CS 架构、客户端模块、服务端模块、通信协议和数据库结构。可引用 `docs/system_design.md` 中的架构图。

## 5. 详细实现

### 5.1 客户端

说明端口监听、事件记录、本地队列、心跳、断线补发、配置内置方式。

### 5.2 服务端

说明 TCP 接入、节点管理、SQLite 存储、Web 管理台、告警通道、客户端生成工具。

### 5.3 加密通信

说明长度前缀帧、随机 nonce、HMAC-SHA256 完整性校验和共享密钥。

## 6. 测试过程

记录单元测试、烟测、普通端口访问捕获、断线补发、告警测试和兼容性验证结果。
